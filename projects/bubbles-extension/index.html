Install instructions ↓

Install the extension

  1. Open Chrome and go to chrome://extensions
  2. Turn on Developer mode (top-right)
  3. Click Load unpacked
  4. Select this project folder: projects/bubbles-extension (the folder that contains manifest.json)

After that, the Bubbles icon appears in your toolbar. Click it on any webpage and hit Play.
