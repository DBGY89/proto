# Movie Palette

Pick a film from a list of 10 iconic movies and see its color palette. Copy any hex value with one click.

## Problem

Designers and developers often want to reuse the visual mood of a film — but finding or defining a cohesive palette is tedious.

## Hypothesis

A minimal tool that maps famous films to 5-color palettes (with copyable hex) will be useful and delightful.

## Users

Designers, front-end devs, and anyone who loves film and color.

## Metrics

- Time to first palette view
- Number of copy actions per session
