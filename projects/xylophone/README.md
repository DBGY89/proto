# K-Pop Xylophone

Interactive xylophone that **plays the song for you** and lights up the keys in sync—so you see exactly which key to tap, when. Pick a K-pop tune, watch the demo, then try it yourself. No sheet music, no judgment.

## Songs

- **BTS:** Dynamite, Butter, Boy With Luv
- **BLACKPINK:** How You Like That, DDU-DU DDU-DU, Kill This Love
- **Red Velvet:** Psycho
- **iKON:** Love Scenario

## How it works

1. Choose a song from the dropdown.
2. The app plays the melody and **highlights each key** as it sounds—interactive instructions.
3. When it’s over, it’s your turn: tap the keys in the same order, or hit **Replay** to watch again.
4. Keys 1–8 = C, D, E, F, G, A, B, C′ (one octave).

Run a local server from the repo root (`python3 -m http.server 8080`) and open the project from the lab.
